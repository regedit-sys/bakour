```
gpg --recv-keys gpg --recv-keys 410B4A906DF63761AA2B26DC2C316AF89E7074F3
```
in your terminal to recieve my GPG Signing Key!

Now add the pacman repo by putting
```
[bakour]

```
in your /etc/pacman.conf file.
